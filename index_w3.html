Projects

Responsive Restaurant Menu Site

A fully responsive, multi-page website built for a local restaurant. The project uses semantic HTML5 for structure and CSS media queries to adapt the layout across mobile, tablet, and desktop viewports. The navigation collapses into a vertical single-column list on small screens and expands to a horizontal bar on wider viewports.

Technologies used: HTML5, CSS3, media queries

Task Tracker Web App

A single-page task management application built with React and a Node.js back end. Users can add, edit, delete, and filter tasks by status. The application communicates with a local Express REST API and persists data using JSON file storage. The front end uses functional components and React Hooks exclusively, with no class-based components.

Technologies used: React, Node.js, Express, HTML5, CSS3

View Repository on GitHub

Personal Budget Dashboard

A browser-based budget tracking tool that reads transaction data from a CSV file and displays a summary of income versus expenses by category. Built with vanilla JavaScript and HTML5 table elements, with no external libraries. The user can upload a CSV, and the dashboard calculates totals and renders the results in a structured table.

Technologies used: HTML5, CSS3, JavaScript (ES6+)
